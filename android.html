Temperature Converter Android Application

Objective
This is an elementary and straightforward Temperature converter android application which is used to convert temperature from Celsius to Fahrenheit or vice versa. Functionality includes how to drag and drop into a layout view, to add User Interface (UI) components to the view, add/set properties for your components as well as manually add and edit various filesControls for this app include edittext, button, radiogroup, radio buttons and an imageview.

Note making Application

Objective
This is a single page HTML5 application for taking a list of notes. This single HTML page will open in the browser and all interactions will happen in the same page with JavaScript, the browser itself should never have to refresh the page. The data from the note list will be persistent between loads using local browser storage.

Working
When application opens in the browser, user is presented with ADD NOTE section and empty VIEW NOTE(S) section on the screen. User enters details on Add note section to add a note in the NoteList and click on "Add Note" button.
The user can view the added note in the VIEW NOTE(S) with note id and other note details.If user want to view the note details entered, user clicks on the Subject link of that particular note.
The user can view the entire note details including name, date, time of note creation, subject, note message and character count of note message.
If user wants to delete the created note, then user clicks on "Delete Note" button to delete that particular note from the notelist.
